# PixivDL Browser v0.5.6

PixivDL Browser v0.5.6 是 bug 修复版本，修复了批量下载、收藏夹备注和队列渲染中的关键问题。

## 修复内容

- 修复收藏夹备注 textarea 在移除或切换收藏时的定时器内存泄漏
- 修复批量下载过程中进度条污染工作区 UI 状态的问题
- 修复批量下载队列 DOM 复用逻辑，改用 PID 映射而非脆弱的位置索引

## 变更

- 将项目本地 skill 重命名为可复用的 `agent-collaboration-framework`
- 新增 `docs/CODE_WIKI.md` 文档

## 适合谁

- 已经在 Chrome 或 Edge 中登录 Pixiv 的用户。
- 使用批量下载功能下载多个收藏作品的用户。
- 在收藏夹详情面板中为作品添加备注的用户。

## 主要功能

- 按 PID 获取 Pixiv 插画/漫画作品信息。
- 弹窗按作品预览、图片选择、浏览器下载三栏组织。
- 支持 ZIP 下载和逐张原图下载。
- 批量下载收藏夹中的所有作品，顺序处理避免限流。
- 收藏夹备注支持防抖自动保存。
- 本地代理设置，支持 HTTP、HTTPS、SOCKS4 和 SOCKS5。
- 收藏仓库支持搜索、导入/导出 JSON。

## 安装

1. 下载 `PixivDL-Browser-0.5.6.zip`。
2. 解压 ZIP 文件。
3. 为了保留收藏和缓存，建议把内容覆盖到固定目录，例如 `PixivDL-Browser-latest`。
4. 打开 Chrome 或 Edge 的扩展管理页。
5. 开启"开发者模式"。
6. 点击"加载已解压的扩展程序"，选择固定目录。
7. 后续升级时继续覆盖同一目录，并在扩展管理页点击刷新。

## 限制

- 只支持 Pixiv 静态插画和漫画图片，不支持 ugoira 动图。
- Pixiv 网页 AJAX 接口不是公开稳定 API，Pixiv 改站后可能需要维护。
- 本地文件夹写入需要用户通过浏览器文件夹选择器授权。
- 代理设置属于浏览器扩展作用域。
- 只应下载当前 Pixiv 账号有权限访问的内容，并遵守 Pixiv 及作者的使用规则。
